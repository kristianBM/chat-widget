Embeddable Chat Widget

A production-quality Intercom-style chat widget built with Preact + Vite. One script tag. No dependencies.

⚡

Single Script Embed

Drop one <script> tag into any page. CSS is auto-injected. No configuration of stylesheets needed.

🎨

Fully Themeable

CSS custom properties control every color, radius, and position. Switch themes instantly at runtime.

🔗

n8n Compatible

Works out of the box with n8n webhook nodes. Also handles custom response formats automatically.

💾

Session Persistence

Conversations persist across page reloads via localStorage. New conversation resets cleanly.

📱

Mobile Ready

Full-screen mode on small screens. Touch-friendly targets. Smooth spring animations throughout.

🌙

Dark Mode

Follows system preference by default. Can be forced light or dark via the darkMode option.

Try Different Themes

Click a theme to re-initialize the widget with a new color. The bubble in the corner will update instantly.

Embed snippet 
<!-- 1. Load the widget script (just one tag!) -->
<script src="https://cdn.example.com/chat-widget.umd.js"></script>

<!-- 2. Initialize with your config -->
<script>
  ChatWidget.create({
    webhookUrl: 'https://your-n8n.app/webhook/xxxx',
    title: 'Support',
    subtitle: "We're here 24/7",
    botName: 'Assistant',
    initialMessages: ['Hello! 👋', 'How can I help?'],
    persistSession: true,
    theme: {
      primaryColor: '#6366f1',
      position: 'bottom-right',
      darkMode: 'auto',
    },
  });
</script>

Response Formats

The widget normalises { output }, { text }, { message }, { response }, { reply }, arrays, and plain strings automatically.

Public API

ChatWidget.create(config) mounts the widget. ChatWidget.destroy() removes it cleanly. Safe to call multiple times.

Callbacks

Hook into onOpen, onClose, onMessage, and onError for analytics, custom logic, or integrations.

Custom CSS

Pass a customCSS string to inject additional styles scoped inside the widget's shadow-like .cw-root container.
